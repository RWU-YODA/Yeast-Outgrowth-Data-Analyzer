New Experiment

To upload a new stain click the home page then the new experiment button. This will bring you to the new experiment page. On this page enter the information in the top box. Then click the file upload button and upload the necessary files, these files need to be COMMA DELINIATED. enter the remianing information required in the box from top to bottom. THIS IS IMPORTANT TO HAVE TI AUTO FILL DATA FOR YOU! The files need to have a time column, n columns of data and a column labeled blank at the end. the number of columns of data needs to be divisable by the replication.
